## Käyttöohje
Ohjelma käynnistyy DDSheet-hakemistossa komennolla

```
mvn compile exec:java -Dexec.mainClass=ddsheet.Main
```

tai jar-tiedostolla komennolla

```
java -jar DDSheet-1.0-SNAPSHOT.jar 
```
